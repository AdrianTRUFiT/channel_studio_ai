# Studio Channel – Complete Simple Team Workflow

*(Kid-friendly system so 10-year-olds know exactly what to do, when, and what they need. “Failing to plan is planning to fail” – this is your plan!)*

**Channel Goal**: Make educational faceless YouTube videos (e.g., AI Governance – Authorization) + earn revenue with honest affiliate links.

**1. Folder Structure (Set This Up First on Computer)**

Create these folders once (use D:\Studio or similar):

- **D:\Studio\Instructions** ← Put this guide + request notes here
- **D:\Studio\Scripts** ← Video scripts
- **D:\Studio\Visuals** ← Pictures, clips, thumbnails
- **D:\Studio\Voiceovers** ← Audio files
- **D:\Studio\Videos** ← Edited MP4 files (raw)
- **D:\Studio\Output\Completed** ← Finished uploaded videos + links
- **D:\Studio\Affiliates** ← Your affiliate links + disclosures
- **D:\Studio\Journal** ← Review sheets & progress tracking

**2. Team Roles (Rotate Every Video So Everyone Learns Everything)**

Assign 1–2 kids per role per video:

- **Project Manager** – Keeps checklist on track, moves files between folders
- **Researcher/Writer** – Script + affiliate research
- **Visuals Hunter** – Finds pictures/clips/thumbnail
- **Voice/Editor** – Makes voiceover + builds video in Canva/CapCut
- **SEO & Affiliates** – Title, description, tags, links
- **Reviewer** – Final checks before upload

**3. Full Video Production Timeline (One Video Takes ~1 Week)**

**Day 1: Planning & Script**

- Read the Request (from Instructions folder)
- Write script (250–400 words) + note places for visuals + 1 affiliate mention
- Save script to Scripts folder
- Researcher finds 1–2 good affiliate products

**Day 2: Gather Materials**

- Visuals Hunter collects 10–20 images/clips → Visuals folder
- Voice kid records or makes AI voiceover → Voiceovers folder
- Thumbnail designer starts work in Canva

**Day 3: Build the Video**

- Editor puts it together in chosen tool (Canva/CapCut/PowerPoint)
- Add voice, text on screen, music, transitions
- Export MP4 to Videos folder

**Day 4: Pre-Launch Prep**

- SEO kid writes Title, Description (with affiliate links + disclosure), Tags, Timestamps
- Upload thumbnail
- Reviewer watches full video + checks everything

**Day 5: Upload & Go Live**

- Uploader follows YouTube Studio steps (from previous checklist)
- Set to Public or Schedule
- Pin affiliate comment
- Save all files to Output\Completed

**Day 6–7: Review & Improve**

- Whole team watches published video
- Fill Journal sheet: What worked? What to improve?
- Plan next video

**4. Master Checklist (Print This & Put on Wall)**

Use this every video. Check off with stickers.

**Phase 1: Instructions & Script**

- Read Request
- Write script + affiliate idea
- Save files in correct folders

**Phase 2: Make Video**

- Gather visuals
- Record voiceover
- Build & export video
- Create thumbnail

**Phase 3: Launch**

- Write Title + Description (with disclosure)
- Add tags, cards, end screen
- Upload & set Public
- Pin comment & share link

**Phase 4: After Launch**

- Journal review
- Check analytics after 1 day

**5. Weekly Team Meeting (15 minutes)**

- Sunday or Monday:
    - Review last video results
    - Pick next topic from Instructions
    - Assign roles for new video
    - Celebrate wins (even small ones!)

**6. Tools & Skills Needed (Keep Simple)**

**Main Tools** (all free/easy):

- Canva or CapCut (video + thumbnail)
- Google Docs (scripts)
- YouTube Studio (upload)
- TradingView / Finviz (if doing any trading content later)
- Affiliate dashboards (Amazon, etc.)

**Skills Everyone Practices**:

- Following instructions
- Writing simple scripts
- Finding good pictures
- Honest affiliate recommending
- Checking work (quality control)

**7. Tracking Progress (So Everyone Sees Success)**

- Big wall chart with videos completed (use stickers)
- Simple Google Sheet or notebook in Journal folder:
    - Video Title | Date Uploaded | Views | Affiliate Clicks | Lessons Learned
- Monthly goal: “Finish 4 videos” or “Improve one thing each video”

**8. Safety & Money Rules**

- Adult supervises all accounts, YouTube, affiliate sign-ups
- Always disclose affiliates
- Start with paper trading if mixing in day trading content later
- Focus on learning first – revenue grows over time
- Follow YouTube rules (no spam, honest content)

This system is **repeatable** and **visual**. Once set up, kids can run it mostly on their own with the checklists.

# S**tudio Channel – Projects & Tasks**

**One-Time Setup Project (Do This Once)**

**Goal**: Get the whole system ready so every video is easy.

**Tasks** (check off with stickers):

1. **Create Folders** – Project Manager + Adult
    - Make D:\Studio with subfolders: Instructions, Scripts, Visuals, Voiceovers, Videos, Output\Completed, Affiliates, Journal.
    - Save this guide in Instructions folder.
2. **Set Up Accounts** – Adult only
    - YouTube channel named “Studio” (or whatever name)
    - Sign up for 1–2 affiliate programs (Amazon, Canva, etc.)
    - Save login info safely
3. **Make Team Role Cards** – Everyone
    - Write names on cards: Project Manager, Researcher/Writer, Visuals Hunter, Voice/Editor, SEO & Affiliates, Reviewer.
4. **Print Checklists** – Project Manager
    - Print this whole page + the Master Checklist (below).

**Done?** → Have a team celebration and move to Video Project.

**Main Project: Make One Faceless YouTube Video**

**Goal**: Finish a 2–5 minute educational video (example: AI Governance – Authorization) with affiliate links and get it live.

**Phase 1: Planning (Day 1)**

**Tasks**:

- Read the Request note (Researcher/Writer)
- Write short script (250–400 words) + mark spots for pictures + 1 affiliate idea (Researcher/Writer)
- Save script to Scripts folder
- Pick 1 good affiliate tool that matches the topic (Researcher/Writer)

**Phase 2: Gather Stuff (Day 2)**

**Tasks**:

- Find 10–20 free pictures/clips → save to Visuals folder (Visuals Hunter)
- Make or record voiceover → save to Voiceovers folder (Voice/Editor)
- Start thumbnail in Canva (Visuals Hunter)

**Phase 3: Build Video (Day 3)**

**Tasks**:

- Put pictures + voice + text together in Canva or CapCut (Voice/Editor)
- Add simple music and transitions
- Export as MP4 → save to Videos folder
- Finish thumbnail → save to Visuals folder

**Phase 4: Get Ready to Launch (Day 4)**

**Tasks**:

- Write Title, Description (with affiliate link + disclosure), Tags, Timestamps (SEO & Affiliates)
- Reviewer watches full video and says “Good” or “Fix”
- Save all final files

**Phase 5: Upload & Go Live (Day 5)**

**Tasks**:

- Upload MP4 in YouTube Studio (Uploader / Project Manager)
- Add thumbnail, description, tags, cards, end screen
- Set to Public (or Schedule)
- Pin a comment with affiliate link
- Save link + files to Output\Completed folder

**Phase 6: Review & Celebrate (Day 6–7)**

**Tasks**:

- Whole team watches the live video
- Fill Journal sheet: What worked? What next? (Everyone)
- Update progress wall chart

**Master Checklist (Use for EVERY Video)**

Glue this on the wall. Check off each day.

**Planning**

- Script done
- Affiliate chosen

**Make Video**

- Visuals collected
- Voiceover ready
- Video built & exported
- Thumbnail done

**Launch**

- Title + Description ready
- Uploaded & Public
- Comment pinned

**Review**

- Journal filled

**Weekly Project: Team Meeting (Every Sunday or Monday – 15 minutes)**

**Tasks**:

1. Look at last video’s Journal notes
2. Pick next Request from Instructions folder
3. Assign roles for the new video
4. Celebrate what you finished!

**How Everyone Knows What to Do**

- **Project Manager** looks at the wall checklist every day and tells the team: “Today is Phase 2 – Visuals Hunter, go find pictures!”
- Move finished files to the right folder so the next person knows what’s ready.
- If stuck → ask the Reviewer or Adult.
- Rotate roles every video so everyone learns all tasks.

This is your full plan:

**Setup Project** (once) → **Video Project** (repeat) → **Weekly Meeting** (every week).
