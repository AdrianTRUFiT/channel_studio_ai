# Research Engine

Below is the Research Engine built the way you build systems:
structured, agent‑driven, pipeline‑controlled, and revenue‑aligned.

This is the engine that feeds your Lead Engine, Content Engine, and Revenue Engine.
It is the first engine in the chain because everything downstream depends on the quality, clarity, and structure of your research.

---

⭐ Core Takeaway

Your Research Engine is a three‑layer hybrid system:

1. Intake Layer — gather raw data
2. Analysis Layer — transform chaos into structured intelligence
3. Synthesis Layer — produce usable insights for leads, content, and offers

Pipelines give you deterministic flow.
Agents give you intelligence and pattern recognition.

---

1. 📥 Intake Layer (Raw → Organized)

This layer collects everything the system needs to understand your market.

Pipeline stages

- research_intake_stage
• data_normalization_stage

Agents

- Intake Agent — pulls sources, scrapes summaries, collects examples
• Normalization Agent — cleans, formats, and standardizes data

Inputs

- competitor pages
• audience posts
• testimonials
• product pages
• industry reports
• social content
• search queries

Outputs

- /research/raw/
• /research/normalized/

This layer ensures you never start from a blank page again.

---

1. 🔍 Analysis Layer (Organized → Insight)

This is where the system thinks.

Pipeline stages

- pattern_extraction_stage
• cluster_analysis_stage
• signal_detection_stage

Agents

- Pattern Agent — extracts repeated pains, desires, objections
• Clustering Agent — groups insights into themes
• Signal Agent — identifies buying triggers and urgency signals

Outputs

- patterns.json
• clusters.json
• signals.json

This layer gives you the truth of the market, not guesses.

---

1. 🧠 Synthesis Layer (Insight → Action)

This is where research becomes usable intelligence for the rest of the system.

Pipeline stages

- insight_synthesis_stage
• ICP_generation_stage
• opportunity_map_stage

Agents

- Synthesis Agent — turns patterns into narratives
• ICP Agent — generates ideal customer profiles
• Opportunity Agent — identifies gaps, angles, and offer opportunities

Outputs

- [insights.md](http://insights.md/)
• icp_profiles.json
• opportunity_map.md

This layer produces the fuel for your Lead Engine, Content Engine, and Offer Engine.

---

🧩 Full Research Engine Architecture

Layer	Pipeline Role	Agent Role	Output	
Intake	deterministic	intake + normalization	raw → clean data	
Analysis	deterministic	pattern + cluster + signal	structured intelligence	
Synthesis	deterministic	synthesis + ICP + opportunity	actionable insights

This is the hybrid model:
pipelines control the flow, agents handle the thinking.

- research/raw — unprocessed data
